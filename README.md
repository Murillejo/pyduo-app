# 🧪 Pyduo

## 🚀 ¿Qué es?

Pyduo es una aplicación web creada para aprender Python de forma sencilla, práctica e interactiva.

---

## 🎯 Objetivo

El objetivo de Pyduo es facilitar el aprendizaje de programación, especialmente para personas que empiezan desde cero.

---

## 💡 Idea del proyecto

Pyduo nace como una herramienta educativa que busca hacer el aprendizaje de Python más accesible, guiado y fácil de entender.

---

## 🛠️ Estado del proyecto

⚙️ En desarrollo y mejora continua

---

## 🧠 Enfoque

- Aprendizaje práctico
- Explicaciones simples
- Experiencia amigable para principiantes
